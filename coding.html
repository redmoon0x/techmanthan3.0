Instructions

  1. Only two participants are allowed in a team.
  2. Programming language :C
  3. Cheating in any form will result in immediate disqualification.
  4. Judges decision will be considered as final.

Round Details

  1. Quiz (5 minutes):

    Participants will face a quick quiz to test their knowledge of programming concepts, algorithms, and data structures

  2. Problem Solving (15 minutes) :

    Participants will be presented with coding problems to solve within a given time frame. The problems may vary in difficulty and complexity

  3. Blind Coding (30 minutes):

    Participants will be given a coding task on the spot and to be solved in a given time frame. No visible display during the test. Marks will be allotted based on the understanding of the problem statements (A paper will be given to write the code first.)

Judging Criteria

  1. Problem Solving.
  2. Logical Thinking.
  3. Syntax.
